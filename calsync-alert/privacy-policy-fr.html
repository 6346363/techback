Politique de Confidentialité de CalSyncAlert

CalSyncAlert (ci-après dénommée « cette application ») s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous traitons les informations des utilisateurs.

1. Informations Auxquelles Nous Accédons

Cette application accède aux données de votre calendrier iOS pour fournir une fonctionnalité d'alarme basée sur vos événements programmés. Cet accès est essentiel pour la fonctionnalité principale de l'application.

Important : Toutes les données de calendrier auxquelles cette application accède restent uniquement sur votre appareil. Nous ne collectons, ne stockons ni ne transmettons aucune de vos informations de calendrier à des serveurs externes.

2. Collecte d'Informations Personnelles

Cette application ne collecte ni ne stocke aucune information personnelle des utilisateurs. L'application ne recueille pas de données utilisateur telles que les noms, adresses e-mail, informations de localisation ou statistiques d'utilisation.

3. Utilisation des Données de Calendrier

Les données de calendrier sont utilisées exclusivement pour :

  • Lire les horaires des événements pour définir les alarmes appropriées
  • Afficher les informations des événements dans l'application

Ces données sont traitées localement sur votre appareil et ne sont jamais transmises via Internet.

4. Partage de Données avec des Tiers

Cette application ne fournit pas de données utilisateur ni d'informations de calendrier à des tiers. Aucune communication de données n'a lieu via Internet.

5. Stockage des Données

Toutes les données générées par l'application, y compris les paramètres d'alarme, restent uniquement sur votre appareil. Nous ne maintenons aucune base de données externe ni stockage cloud pour les données utilisateur.

6. Droits des Utilisateurs

Vous pouvez révoquer l'accès au calendrier à tout moment via les paramètres iOS. Comme nous ne collectons pas d'informations personnelles en externe, les questions concernant la suppression de données de nos serveurs ne s'appliquent pas.

7. Mesures de Sécurité

Cette application fonctionne uniquement dans un environnement local sur votre appareil iOS et ne transmet pas de données à des serveurs externes. Vos données de calendrier sont protégées par les fonctionnalités de sécurité intégrées d'iOS.

8. Confidentialité des Enfants

Comme cette application ne collecte aucune information personnelle en externe, elle est sûre pour les utilisateurs de tous âges, y compris les enfants de moins de 13 ans.

9. Modifications de la Politique

Nous nous réservons le droit de mettre à jour cette politique de confidentialité si jugé nécessaire. Toute mise à jour sera publiée sur cette page.
